## Structure

```
> 2022

```

## Running the code

With node installed:

```bash
$ node 2022/day-3.js
> 7742 # part 1 solution
> 2276 # part 2 solution
```

Or for typescript files:
```bash
$ cd 2025
$ node ./day-1.ts --experimental-strip-types
> 123 # part 1 solution
> 789 # part 2 solution
```
